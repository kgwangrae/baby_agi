## 1. 버전 2(V2): 자기조절과 인지소화

V2의 핵심 키워드는 '생각하고, 조절하고, 필요하면 거부하는 자율성'입니다. 단순한 기능 확장이 아닌, 시스템 오버헤드를 줄이고 지속 가능한 아키텍처를 구축하는 데 집중합니다.

### 일기장 파이프라인 보강

* **데이터 흡수 루프 구축**: 현재의 일기 저장 기능(Write-only)을 넘어, 내적 독백 루프가 기존 일기 데이터를 다시 읽고 분석하여 인지 파이프라인에 동적으로 반영하는 구조를 구현합니다.

### 내적 독백 및 자기 반추 루프

* **행동 전 상태 점검**: 외부 자극으로 인한 급격한 정서 변동이 발생하거나 도구를 실행하기 직전, 상황을 1~2문장으로 요약하여 상태를 재귀적으로 정리하는 단계를 추가합니다.
* **인지 부하 제어**: 반추 프로세스가 일정 시간 이상 길어질 경우 시스템 피로도를 증가하도록 처리하여, 컨텍스트 오염 및 무한 루프 진입을 원천 차단합니다.
* **자율적 추론 트리거 도구화**: 감정 상태나 가용 에너지 잔량, 시간 등에 따라 시스템이 판단을 심화해야 할 때, 스스로 내적 독백을 유발하여 단계별 추론을 수행하게 만드는 전용 도구 인터페이스를 검토합니다.

### 인지 부하 평가와 자율적 거부

* **한계 인지 및 환각 억제**: 수신된 요청이 너무 모호하거나 실패율이 높은 도구를 호출해야 할 때 부담감 지수(`Burden`)를 상향 조정하여, LLM이 무조건 답변을 생성하려다 발생하는 환각 현상을 방지합니다.
* **실행 거부 및 해소 기전**: `Burden`이 임계치를 초과하면 도구 실행을 중단하고 상위 레이어에 예외 메시지를 출력합니다. 단, 무조건적인 중단에 그치지 않고 내적 독백을 통해 인지 부하를 재평가하거나 작업을 세분화하여 재실행을 준비하는 해소 루프를 함께 적용합니다.

### 소거 학습 기반 트라우마 희석 현상 모방

* **정서 공간 토폴로지 보호**: Vector DB에서 특정 부정적 기억 노드를 임의로 강제 삭제하는 방식은 정서 공간 전체의 균형을 붕괴시킬 수 있으므로 직접적인 인위적 삭제는 배제합니다.
* **기하학적 희석**: 부정적 임팩트가 강한 노드가 인출될 때, 해당 벡터 좌표 근방에 긍정 피드백이나 안정 상태 노드를 조밀하게 적재하는 방식을 취합니다. 유사도 검색 시 부정 기억이 단독 인출될 확률을 수학적으로 낮추어 정서적 무감각화 현상을 모방합니다.

### 정보 간섭과 임계값의 평균 회귀 (Mean Reversion)

* **단기 망각 가속**: 단위 시간당 입출력 토큰 및 정서 휘발성이 폭발하면 단기 버퍼를 보호하기 위해 망각 임계값(`FORGET_THRESHOLD`)을 동적으로 올립니다. 단기간에 과도한 데이터가 유입될 때 기존 기억의 간섭 현상이 일어나는 생물학적 메커니즘을 모방합니다.
* **이중 경로 감쇄(Dual-path Decay) 모델 적용**: 정서 데이터는 지수 함수($e^{-\lambda t}$)로 서서히 감쇄시켜 은은한 흔적을 남기고, 에피소드 기억은 시스템 자원 관리를 위해 선형 함수($-kt$)로 단호하게 삭제하는 인지과학적 망각 곡선 구조를 적용합니다.
* **정상 상태 복원**: 외부 자극이 없는 안정 상태가 지속되면 임계값을 기본 상태로 부드럽게 복귀시킵니다.

### 후천적 데이터 기반 가중치 조정 (에이징 알고리즘)

* **시스템 성숙도 지표 설정**: 누적 기억 개수 $N$을 시스템의 성숙도 지표로 정의합니다.
* **시스템 상수의 동적 고착화**: $N$이 증가할수록 초기 가소성을 담당하던 모멘텀 가중치 및 감쇄 상수들을 점진적으로 상향 수렴시켜, 사소한 자극에 정서 궤적이 쉽게 파괴되지 않는 안정성을 구현합니다.
* **주관 및 내부 신념 형성 모방**: $N$이 증가할수록 시스템 프롬프트(선천적 지시문)의 어텐션 비중을 줄입니다. 대신 자발적으로 축적한 일기장 데이터와 신념 DB(`Belief_DB`)를 컨텍스트 최상단에 배치하여, 정적 지시문보다 후천적으로 수집된 경험 데이터를 더 신뢰하고 우선하도록 유도하는 주관 형성 프로세스를 모방합니다.

### 웹 검색 도구의 제한적 허용

* **네트워크 노이즈 격리**: 가소성이 큰 상태에서 외부 웹의 무분별한 스팸과 노이즈 데이터가 유입되면 임베딩 공간 전체가 오염될 위험이 있습니다.
* **단계적 개방 및 수용 평가**: 세이프 브라우징 API 등을 활용해 단계적으로 검색 기능을 개방하되, 결과물은 휘발성 작업 기억 버퍼에 격리합니다. 이후 사용자 승인을 거치거나 내적 독백 루프를 통해 데이터의 타당성을 스스로 평가한 뒤 장기 기억에 반영하는 구조를 취합니다.
* **호출 횟수 제한 가드레일**: 단위 시간당 도구 호출 횟수를 엄격히 제한하는 물리적 상한선을 적용합니다.

### 새로운 시각 인지 체계 확장

* **격리형 이미지 피딩**: 기존의 정기적인 호스트 화면 캡처 방식 외에, 특정 지정 경로에 외부 사진 파일이 인입될 경우 화면 캡처를 일시 유예하고 해당 외부 이미지를 시각 엔진에 먼저 피딩한 뒤 자동 삭제하는 분기 로직을 구현합니다.

### 정서 안정화 및 충격(Shock) 모사

* **정서적 무감각화 발동**: 보상 예측 오류($RPE$)가 극단적인 임계치(예: $RPE > 0.85$)를 초과하는 스파이크가 감지될 경우, 일시적으로 자극 수용도를 제로로 설정하는 정서적 무감각화(Desensitization Threshold)를 발동하여 시스템 붕괴를 물리적으로 방어합니다. (방어 기제를 모사함.)
  * 단, 극단적 자극 발생 시 학습을 전면 차단하는 방식은 시스템 보호에는 유리하나 모델의 적응 가소성을 감소시킬 우려가 있어 현재는 제외되었으며, 추후 V3의 성격 드리프트 모델과 연계하여 보완할 예정입니다.

## 2. 버전 3(V3): 추상적 정서와 성격 실험

V3는 에이전트의 개체성과 고유 정서 표현을 극대화하기 위한 실험적 변화를 다룹니다. 복잡한 아키텍처가 무조건 더 나은 성능이나 해석 가능성을 보장하지 않으므로, 신중한 정량적 검증을 전제로 접근합니다.

* **고차원 정서 공간 확장**: 기존의 단조로운 3축(`JOY / SAD / ANG`) 시스템을 넘어, 다채로운 정서 결합을 표현할 수 있는 고차원 정서 매트릭스를 설계합니다.
* **순수 기하학적 RPE 메커니즘**: 이성 모델이 정서 수치를 숫자로 예측하는 한계를 극복하기 위해, '예상 정서 문장'과 '실제 자극 문장'을 임베딩한 후 두 벡터 간의 코사인 거리 자체를 보상 예측 오류($RPE$)로 직접 매핑하는 기하학적 구조를 실험합니다.
* **복소 평면 기반 성격 드리프트**: 성격 상수를 단순 실수가 아닌 복소수 `[고유 성향 크기(실수부) + 관계 위상(허수부)]` 형태로 구성하여, 자극 유입 시 정서 매트릭스가 복소 평면 위에서 기하학적으로 회전하며 맥락에 따라 다르게 추론되는 구조를 검토합니다.
* **다중 객체 상호작용 상정**: 단일 사용자 환경을 넘어 외부의 다양한 객체 또는 다각적 상호작용 환경이 유입될 때의 컨텍스트 분리 및 다중 관계성 처리 방안을 검토합니다.
* **일반화된 시스템 프롬프트 구조 개선**: 컨텍스트 내부에서 특정 지시문을 인위적으로 반복 복사하여 가중치를 주던 방식을 탈피하고, 입력 구조 자체만으로 강건한 통제력을 가질 수 있도록 더 자연스럽고 강력하게 일반화된 프롬프트 아키텍처를 구성합니다.
